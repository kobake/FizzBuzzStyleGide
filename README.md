Fizzbuzz Style Gide
===================

I love FizzBuzz :).

Rule
====

* Input program integer each number from 1 to 100
  * if n can be devided by 3, output "Fizz"
  * if n can be devided by 5, output "Buzz"
  * if n can be devided by 15, output "FizzBuzz"
  * other, output n.
